Desenvolvimento

- usar o maven install para compilar a aplicação
- subir como Spring Boot App
- Base de dados está em H2 (memória)
- Arquivo de dados para desenv: data.sql
- Logar como admin: admin@magalu.com.br senha 123456
- Logar como cliente: cliente@magalu.com.br senha 123456
- Verifcar documentação dos controllers da api: localhost:8080/swagger-ui.html
- Página inicial localhost:8080/index.html

Produção
- Acessar https://magalu-finder-otavio.herokuapp.com/index.html
- Na tela de cadastrar Produto adicione os dados e selecione uma loja pre existente para relacionar ambos
- Na pesquisa preencher a descrição ou código e a origem, o valor só é buscado se estiver igual ao cadastrado.
